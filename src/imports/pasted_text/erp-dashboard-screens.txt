Create two modern ERP dashboard screens in Figma for a Hearing Aid Labs management system: one for “Sales Management” and one for “Purchase Management”. The design should take inspiration from an existing old-style ERP dashboard but must be completely redesigned with a modern SaaS admin dashboard approach. Keep all important business workflows, filters, tables, actions, and navigation structure, but improve the visual design, spacing, usability, hierarchy, and component consistency.

The UI should feel premium, enterprise-ready, and scalable. Avoid outdated boxed layouts and dense table styling. Use a clean card-based structure, modern typography, better spacing, rounded corners, soft shadows, and improved readability. Use a desktop-first layout designed around 1440px width.

Use a professional admin dashboard aesthetic similar to modern ERP systems like Stripe Dashboard, Linear, Notion Admin, Odoo modern redesigns, or enterprise SaaS tools.
